## 安裝
請先裝`nodejs 6`
```bash
cd tau3_phah4-ji7/
npm i
npm start 

#瀏覽 http://localhost:3000/
```

## 排版
```bash
npm run check #檢查排版
npm run reformat #自動排版
```
本專案用`eslint`排版

## 佈署上線
`npm run deploy`
